# MedDiagno

MedDiagno is a machine learning–based medical diagnosis project that uses deep learning models
to assist in disease prediction (e.g., skin cancer detection).

## Features
- Deep learning models built using TensorFlow/Keras
- Medical image–based disease prediction
- Large model handling using Git LFS
- Backend-ready for Flask/FastAPI integration

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Git LFS
